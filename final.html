Password Strength Checker and Generator

Yat Long Chan, Oliver Luo, Samson Wu, Period 4

Project Description

This is a program that will check the strength of a password as well as generate a strong password.

Instructions

  1. Install the Python program and text files in the same directory.
  2. Run the program using a Python IDE.
  3. Follow instructions displayed in the console.

How it works

The password checker part of this program works by using a strike system. The number of strikes determines the strength of the password. The number of strikes build up when the inputted password is run through a series of checks. If the password is less than 12 characters a strike is added. If the password does not contain uppercase letters, lowercase letters, special characters, or numbers a strike is added for each one missing. 0 strikes makes a strong password, 1-2 strikes make a medium password, 3 strikes make a weak password, and 4 or more strikes make a very weak password. There are other checks like if the password uses a dictionary word with numbers or if it is just one character repeating. These other checks will return the other statements that point out the problem with the password. Example of how the password checker works:

The generator part of this program works by simply appending random characters of all types to an empty list. The length of the list in the end will be the length that the user inputted. This list will be shuffled to create the generated password. The generator will not generate any passwords under the length of 12 or over the length of 100. Example of how password generator works:

Files

  • final.html: This is the html file that contains the documentation of our project
  • 10k-most-common.txt: This file contains a list of the most common words and number combinations.
  • words_alpha.txt: This file contains a list of English dictionaries words.
  • passchecker.py: This is the python program that checks whether your inputted password is strong and generates a random password that is strong.

  • The list of words includes singular letters, so typing one letter counts as a dictionary word.
